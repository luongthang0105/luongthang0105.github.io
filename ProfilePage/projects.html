Thomas's Bizarre Projects

Finding Dory

  • The project was created using Python, Matplotlib library, and some abnormal home-made algorithms.
  • The main function of Finding Dory is to find the shortest path between 2 selected points and visualize it on the given maze.
  • Specifically, I used the Dijkstra's algorithm to find the shortest path, and the Ray Casting algorithm to detect whether a point lies inside the polygon!

Finding Nemo

  • Finding Nemo is an Android application that is created by the Flutter framework and has its data stored in the Firebase database.
  • The application uses a Geolocation API to detect the exact location of the user on the map.
  • Other than that, one can see others' users' location if they add each other as a contact on the app.

Automatic Attendance System (AAS)

  • The most tedious and enormous Arduino project I've ever done!
  • AAS keeps track of the attendance of a class by having the students tap their NFC card onto the Arduino system and recording each attendant on a seperate Excel file.
  • About the hardwares, I used the RFID-RC522 Arduino module to read the students' info from NFC card.
  • On the other hand, whenever the system reads in a NFC card, it sends the received infos to the Excel file via the Parallax Data Acquisition tool (PLX-DAQ).
